GraphIQ is an AI-Powered Relational Intelligence System. Asmith A Bangera, Grahith Kumar B, and Manvith Devadiga are the developers of GraphIQ. They are students at NNM College of Engineering. The project GraphIQ is submitted for Advanced Machine Learning course AM2001-1. GraphIQ is a Knowledge Graph Reasoning Engine. The system uses spaCy and BERT for Named Entity Recognition. GraphIQ uses Neo4j as the graph database. Node2Vec is used for graph embeddings. FAISS is used for vector similarity search. FastAPI provides the backend API. React and Vite are used for the frontend. Groq and Llama3 power the AI reasoning. GraphIQ supports risk analysis and compliance monitoring. The project was submitted in February 2026. GraphIQ uses PyTorch and sentence-transformers for machine learning. The hybrid retrieval combines graph traversal and semantic search. GraphIQ demonstrates explainable multi-hop reasoning.
